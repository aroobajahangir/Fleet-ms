# 🚗 Fleet Management System

A **C# Console Application** for managing a fleet of vehicles, drivers, trips, fuel records, and maintenance activities. This system helps in organizing and tracking all fleet operations efficiently.

---

## 📌 Features

- 🔐 User Login System  
- 🚘 Vehicle Management (Add / View Vehicles)  
- 👨‍✈️ Driver Management (Add / View Drivers)  
- 🛣️ Trip Management (Create & View Trips)  
- ⛽ Fuel Tracking System  
- 🔧 Maintenance Records Management  
- 💾 File Handling / Database Integration  
- 📊 Console-Based Interactive Menu System  

---

## 🛠️ Technologies Used

- C# (.NET)
- Object-Oriented Programming (OOP)
- Console Application
- MongoDB / LocalDB (if used)
- File Handling
- Visual Studio
